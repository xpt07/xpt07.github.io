2D Custom Physics Engine

C++ rigid-body dynamics with collision detection and OpenGL sandbox

C++ Physics Collision Detection OpenGL Premake

Overview

This project implements a 2D physics engine in C++, supporting rigid-body dynamics, collision detection, and impulse resolution. A real-time OpenGL sandbox visualiser was developed to demonstrate simulations of shapes interacting under gravity and user-applied forces. The engine was written from scratch to explore fundamental physics concepts, numerical stability, and efficient collision handling.

Key Features

  • Rigid Body Dynamics: Support for linear and angular motion with configurable mass and inertia tensors.
  • Collision Detection: Broadphase and narrowphase algorithms for efficient shape intersection tests.
  • Impulse Resolution: Conservation of momentum and restitution handling for realistic responses.
  • Constraint Handling: Basic support for joints and contact persistence.
  • Sandbox Application: Interactive OpenGL-based visualiser for debugging and showcasing simulations.
  • Extensible Framework: Modular C++ design with clear separation of physics core and rendering.

Technical Details

  • Written in modern C++17, using object-oriented design for bodies, shapes, and collision manifolds.
  • Matrix and vector math implemented in-house to deepen understanding of transformations and rotations.
  • Collision resolution uses impulses with positional correction to prevent sinking artifacts.
  • Build system configured with premake5.lua, generating Visual Studio project files.

Demo & Instructions

To build and run the engine:

  1. Clone the repo: git clone https://github.com/xpt07/Custom_Physics_Engine.git
  2. Run buildVS2022Projects.bat (or the 2019 script) to generate solution files.
  3. Open the solution in Visual Studio and build the Sandbox project.
  4. Run the executable to launch the interactive demo.
